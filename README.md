## 🔗 [Launch Live Dashboard Matrix](https://ayobro-hash.github.io/mrtlc-nexus/)

Welcome to **MRTLC Nexus v4.0**, a standalone, client-side binary processing deck...


## 🚀 Key Features
* **100% Client-Side:** Processes binary file arrays (`.rbxl` / `.rbxm`) directly inside your browser using fast memory arrays. No slow server uploads required!
* **Mobile-First Pipeline:** Built, debugged, and launched entirely from a mobile device using Termux.
* **Local Testing Harness:** Equipped with an internal Node/Express server for fast, offline layout testing.

## 🛠️ Local Development
If you pull this repository down locally, you can spin up the built-in testing server with:

```bash
# Install local packages
npm install

# Start the local engine
node server.js
```
